soccer.js

ES5, Jasmine, CSS

soccer.js is a simple open source JavaScript library used for quickly building responsive and attractive soccer league tables. The library is built using vanilla JavaScript ES5.

The library provides an easy way to update league tables with added table reverse functionality and an optional dropdown toggle. soccer.js has been built to account for every scenario in mind. Teams, are by default, automatically sorted by points, then by goal difference, goals scored, and lastly, alphabetically.

Geoggle

ES6, Webpack, Sass

Geoggle is a fun and highly addictive countries of the world quiz game. The quiz is built using vanilla ES6 - taking advantage of the new built-in modules feature to help modularise the application. Geoggle is fully responsive and styled using the Sass CSS preprocessor.

The quiz can be played in either 'easy' or - for the geography buffs - 'hard' mode. Questions are based on country flags, capitals, populations, and world regions and subregions. Players receive instant feedback for every question answered, making the application fun as well as educational.

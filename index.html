Personal Projects

soccer.js

ES5, Jasmine, CSS

soccer.js is a simple open source JavaScript library used for quickly building responsive and attractive soccer league tables. The library is built using vanilla JavaScript ES5.

The library provides an easy way to update league tables with added table reverse functionality and an optional dropdown toggle. soccer.js has been built to account for every scenario in mind. Teams, are by default, automatically sorted by points, then by goal difference, goals scored, and lastly, alphabetically.

Geoggle

ES6, Webpack, Sass

Geoggle is a fun and highly addictive countries of the world quiz game. The quiz is built using vanilla ES6 - taking advantage of the new built-in modules feature to help modularise the application. Geoggle is fully responsive and styled using the Saas CSS preprocessor.

The quiz can be played in either 'easy' or - for the geography buffs - 'hard' mode. Questions are based on country flags, capitals, populations, and world regions and subregions. Players receive instant feedback for every question answered, making the application fun as well as educational.

Issue-track

ES5, Handlebars, CSS

Issue-track is an issue/bug tracking application. This is my first JavaScript project. The aim was to create a simple but useful application to help learn the fundamentals of ES5 JavaScript and CSS.

Issue-track has multiple features. The application enables users to submit issues based on their type and also by their severity. The issues can then be filtered by their current status (open or closed) and also by severity. The localStorage API is used to persist issue data.

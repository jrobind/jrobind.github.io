Personal Projects

soccer.js

ES5, JavaScript, CSS

soccer.js is a simple open source JavaScript library used for quickly building responsive and attractive soccer league tables. The library is built using vanilla JavaScript ES5.

The library provides an easy way to update league tables with added table reverse functionality and an optional dropdown toggle. soccer.js has been built to account for every scenario in mind. Teams, are by default, automatically sorted by points, then by goal difference, goals scored, and lastly, alphabetically.

Fit-Search

ES6, React, Redux, CSS, Express, MongoDB

Fit-Search is a personal trainer search and review application. Fit-Search is built with React and Redux using the latest E6S JavaScript, and with an API backend built using Express and MongoDB.

Fit-Search helps people find trusted, high quality, personal trainers across England. The application serves both user and trainer and interests. Whilst users able to exploit the applications unique search and review functionality, trainers are able to use the application to help build a strong client base.

Issue-track

ES5, JavaScript, Handlebars, CSS

Issue-track is an issue/bug tracking application. This is my first JavaScript project. The aim was to create a simple, but useful application to help learn the fundamentals of ES5 JavaScript and CSS.

Issue-track has multiple features. The application enables users to submit issues based on their type and also by their serverity. The issues can then be filtered by their current status (open or closed) and also by severity. The locaStorage API is used to persist issue data.
